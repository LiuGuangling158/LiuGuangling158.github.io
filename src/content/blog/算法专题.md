## 专题 1：排序算法

### 1.1 Python vs Java 内置排序

日常刷题 90% 的情况直接调内置排序即可：

```python
# Python
arr.sort()                          # 原地升序
arr.sort(reverse=True)              # 原地降序
sorted_arr = sorted(arr)            # 返回新列表
arr.sort(key=lambda x: (x[0], -x[1]))  # 自定义排序
```

```java
// Java
Arrays.sort(arr);                              // 升序
Arrays.sort(arr, (a, b) -> b - a);            // 降序（需要 Integer[]）
Arrays.sort(arr, (a, b) -> a[0] - b[0]);     // 二维按第一列
Collections.sort(list, Comparator.comparing(...));
```

Python 的 `sort` 是 **Timsort**（归并+插入的混合），Java 的 `Arrays.sort` 对基本类型用**双轴快排**，对对象用 **Timsort**。

### 1.2 手写快排（面试考点）

```python
def quicksort(arr, left, right):
    if left >= right:
        return
    pivot = partition(arr, left, right)
    quicksort(arr, left, pivot - 1)
    quicksort(arr, pivot + 1, right)

def partition(arr, left, right):
    pivot = arr[right]              # 选最右为基准
    i = left                        # i 指向"小元素区域的最后一个"
    for j in range(left, right):
        if arr[j] < pivot:
            arr[i], arr[j] = arr[j], arr[i]
            i += 1
    arr[i], arr[right] = arr[right], arr[i]
    return i                        # pivot 最终位置
```

```java
void quicksort(int[] arr, int left, int right) {
    if (left >= right) return;
    int pivot = partition(arr, left, right);
    quicksort(arr, left, pivot - 1);
    quicksort(arr, pivot + 1, right);
}

int partition(int[] arr, int left, int right) {
    int pivot = arr[right];
    int i = left;
    for (int j = left; j < right; j++) {
        if (arr[j] < pivot) {
            int t = arr[i]; arr[i] = arr[j]; arr[j] = t;
            i++;
        }
    }
    int t = arr[i]; arr[i] = arr[right]; arr[right] = t;
    return i;
}
```

### 1.3 手写归并排序

归并排序的核心价值不在于排序本身，而在于**分治思想**和**归并过程**——很多算法题（如逆序对、链表排序）都是归并的变体。

```python
def mergesort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = mergesort(arr[:mid])
    right = mergesort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i]); i += 1
        else:
            result.append(right[j]); j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```

### 1.4 计数排序（数据范围有限时 O(n)）

```python
def counting_sort(arr):
    if not arr:
        return arr
    max_val = max(arr)
    count = [0] * (max_val + 1)
    for x in arr:
        count[x] += 1
    result = []
    for val, cnt in enumerate(count):
        result.extend([val] * cnt)
    return result
```

## 专题 2：二分查找

### 2.1 核心模板

二分查找看似简单，但边界条件极易写错。记住下面三个模板就够了：

**模板 1：标准二分（找精确值）**

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:              # 注意：<=
        mid = left + (right - left) // 2  # 防溢出
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

**模板 2：找第一个 ≥ target 的位置（lower_bound）**

```python
def lower_bound(arr, target):
    left, right = 0, len(arr)          # 注意 right 初始化为 len
    while left < right:                 # 注意：<
        mid = left + (right - left) // 2
        if arr[mid] >= target:
            right = mid                 # target 可能在 mid 或左侧
        else:
            left = mid + 1
    return left                         # left = 第一个 ≥ target 的位置
```

**模板 3：找第一个 > target 的位置（upper_bound）**

```python
def upper_bound(arr, target):
    left, right = 0, len(arr)
    while left < right:
        mid = left + (right - left) // 2
        if arr[mid] > target:
            right = mid
        else:
            left = mid + 1
    return left
```

### 2.2 三个模板的关系

```
arr = [1, 2, 2, 2, 3, 4]

binary_search(arr, 2)    → 返回 1/2/3 中的某一个
lower_bound(arr, 2)      → 返回 1（第一个 2）
upper_bound(arr, 2)      → 返回 4（第一个 3 的位置）
```

### 2.3 二分查找的扩展场景

- **旋转数组查找**：`[4,5,6,7,0,1,2]` 中找 target
- **寻找峰值**：数组中比相邻元素都大的位置
- **最小值最大化**：如"分割数组使最大段和最小"（二分答案）
- **第 K 小/大元素**：值域二分

**二分答案模板**（非常重要）：

```python
def binary_answer(arr, k):
    """在 [min_val, max_val] 范围内二分搜索满足条件的最优值"""
    left, right = min(arr), max(arr)
    while left < right:
        mid = left + (right - left) // 2
        if check(mid):      # check(mid) = 在 mid 条件下是否可行
            right = mid     # 可行 → 尝试更小（找最小值）
        else:
            left = mid + 1
    return left
```

## 专题 3：双指针

### 3.1 三类双指针模式

```
┌─────────────────┬─────────────────────┬──────────────────┐
│   对撞指针        │     快慢指针         │     分离指针      │
│ left →  ← right  │ slow →  fast → →    │ i →    j →       │
│                  │                     │                  │
│ 典型场景：        │ 典型场景：           │ 典型场景：        │
│ 有序数组的两数之和 │ 链表环检测           │ 归并两个有序数组  │
│ 三数之和          │ 删除重复元素         │ 两个数组的交集    │
│ 盛水最多容器      │ 移动零              │                  │
│ 反转数组          │ 链表中间节点         │                  │
└─────────────────┴─────────────────────┴──────────────────┘
```

### 3.2 对撞指针模板

```python
def two_sum_sorted(arr, target):
    """已排序数组的两数之和"""
    left, right = 0, len(arr) - 1
    while left < right:
        total = arr[left] + arr[right]
        if total == target:
            return [left, right]
        elif total < target:
            left += 1
        else:
            right -= 1
    return []
```

### 3.3 快慢指针模板

```python
# 删除有序数组中的重复项（原地）
def remove_duplicates(arr):
    if not arr:
        return 0
    slow = 0
    for fast in range(1, len(arr)):
        if arr[fast] != arr[slow]:
            slow += 1
            arr[slow] = arr[fast]
    return slow + 1


# 移动零到末尾
def move_zeroes(arr):
    slow = 0
    for fast in range(len(arr)):
        if arr[fast] != 0:
            arr[slow], arr[fast] = arr[fast], arr[slow]
            slow += 1
```

### 3.4 三数之和（对撞指针 + 去重）

```python
def three_sum(nums):
    nums.sort()
    result = []
    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i - 1]:  # 跳过重复的 i
            continue
        left, right = i + 1, len(nums) - 1
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            if total < 0:
                left += 1
            elif total > 0:
                right -= 1
            else:
                result.append([nums[i], nums[left], nums[right]])
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1
                left += 1; right -= 1
    return result
```

## 专题 4：滑动窗口

### 4.1 核心思想

滑动窗口是双指针的进阶版——维护一个窗口 `[left, right]`，right 扩展寻找可行解，left 收缩优化解。

### 4.2 变长窗口模板

```python
def sliding_window(s):
    """
    寻找满足条件的最长/最短子串
    """
    window = {}       # 窗口内的数据（字符计数、元素集合等）
    left = 0
    result = 0        # 或 float('inf')（找最短时）

    for right in range(len(s)):
        # 1. 将 s[right] 加入窗口
        window[s[right]] = window.get(s[right], 0) + 1

        # 2. 当窗口不满足条件时，收缩左边界
        while not is_valid(window):
            window[s[left]] -= 1
            if window[s[left]] == 0:
                del window[s[left]]
            left += 1

        # 3. 此时窗口是合法的，更新答案
        result = max(result, right - left + 1)

    return result
```

### 4.3 定长窗口模板

```python
def fixed_window(arr, k):
    """长度为 k 的定长窗口"""
    window_sum = sum(arr[:k])
    result = window_sum

    for i in range(k, len(arr)):
        window_sum += arr[i] - arr[i - k]  # 进一个，出一个
        result = max(result, window_sum)

    return result
```

### 4.4 滑动窗口 vs 双指针的区别

| 维度 | 双指针 | 滑动窗口 |
|------|--------|---------|
| 指针移动 | 两个指针各自独立移动 | right 每次 +1，left 按条件收缩 |
| 典型场景 | 有序数组、链表 | 子串、子数组 |
| 窗口维护 | 通常不维护窗口状态 | 维护窗口内的统计信息 |
| 时间复杂度 | O(n) | O(n)（每个元素被访问两次） |

## 专题 5：BFS & DFS

### 5.1 树的遍历（基础）

```python
# DFS 前序遍历（递归）
def preorder(root):
    if not root:
        return
    print(root.val)
    preorder(root.left)
    preorder(root.right)

# DFS 中序遍历（递归）
def inorder(root):
    if not root:
        return
    inorder(root.left)
    print(root.val)
    inorder(root.right)

# DFS 前序遍历（迭代，用栈）
def preorder_iter(root):
    if not root:
        return []
    stack, result = [root], []
    while stack:
        node = stack.pop()
        result.append(node.val)
        if node.right: stack.append(node.right)  # 先右后左
        if node.left:  stack.append(node.left)
    return result

# BFS 层序遍历（用队列）
from collections import deque
def level_order(root):
    if not root:
        return []
    q, result = deque([root]), []
    while q:
        level = []
        for _ in range(len(q)):  # 按层处理
            node = q.popleft()
            level.append(node.val)
            if node.left:  q.append(node.left)
            if node.right: q.append(node.right)
        result.append(level)
    return result
```

### 5.2 图的遍历

```python
from collections import deque, defaultdict

def bfs_graph(graph, start):
    """图的 BFS（最短路径）"""
    q = deque([start])
    visited = {start}
    distance = {start: 0}

    while q:
        node = q.popleft()
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                distance[neighbor] = distance[node] + 1
                q.append(neighbor)

    return distance


def dfs_graph(graph, node, visited=set()):
    """图的 DFS（递归）"""
    visited.add(node)
    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs_graph(graph, neighbor, visited)
```

### 5.3 网格 BFS/DFS

```python
from collections import deque

def bfs_grid(grid, start):
    """网格 BFS（岛屿、迷宫等）"""
    rows, cols = len(grid), len(grid[0])
    q = deque([start])
    visited = {start}
    # 四个方向：上、下、左、右
    directions = [(-1, 0), (1, 0), (0, -1), (0, 1)]

    while q:
        r, c = q.popleft()
        for dr, dc in directions:
            nr, nc = r + dr, c + dc
            if (0 <= nr < rows and 0 <= nc < cols
                    and (nr, nc) not in visited
                    and grid[nr][nc] == 1):  # 可通行条件
                visited.add((nr, nc))
                q.append((nr, nc))

    return visited


def dfs_grid(grid, r, c, visited):
    """网格 DFS（递归）"""
    if (r < 0 or r >= len(grid) or c < 0 or c >= len(grid[0])
            or (r, c) in visited or grid[r][c] == 0):
        return
    visited.add((r, c))
    for dr, dc in [(-1, 0), (1, 0), (0, -1), (0, 1)]:
        dfs_grid(grid, r + dr, c + dc, visited)
```

### 5.4 BFS vs DFS 选型指南

| 场景 | 推荐 | 原因 |
|------|------|------|
| 最短路径（无权图） | BFS | 逐层扩展，第一个到达即为最短 |
| 路径存在性 | DFS（递归简单） | 代码简洁 |
| 拓扑排序 | BFS（Kahn）或 DFS | 各有优劣 |
| 连通分量 / 岛屿数量 | DFS（或 BFS） | 都行 |
| 状态空间搜索 | BFS | 找最短步数 |
| 排列组合 | DFS（回溯） | 需要回溯状态 |

## 专题 6：动态规划

### 6.1 DP 解题五步法

```
Step 1: 定义 dp 含义
   dp[i] 表示什么？dp[i][j] 表示什么？

Step 2: 推导状态转移方程
   dp[i] = ? 依赖于 dp[i-1] / dp[i-2] / ... ?

Step 3: 确定初始条件
   dp[0] = ? dp[1] = ?

Step 4: 确定遍历顺序
   从左到右？从下到上？

Step 5: 确定返回值
   dp[n] / dp[0][0] / max(dp) / ... ?
```

### 6.2 经典 DP 问题汇总

#### 斐波那契型（一维、递推）

```python
# 爬楼梯：每次爬 1 或 2 阶，到第 n 阶有几种方法
def climb_stairs(n):
    if n <= 2:
        return n
    a, b = 1, 2
    for _ in range(3, n + 1):
        a, b = b, a + b
    return b
```

#### 背包型

```python
# 0-1 背包
# dp[i][w] = 前 i 个物品，容量 w 时的最大价值
def knapsack_01(weights, values, capacity):
    n = len(weights)
    # 一维优化
    dp = [0] * (capacity + 1)
    for i in range(n):
        for w in range(capacity, weights[i] - 1, -1):  # 倒序！
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    return dp[capacity]


# 完全背包（每种物品无限个）
def knapsack_complete(weights, values, capacity):
    dp = [0] * (capacity + 1)
    for i in range(len(weights)):
        for w in range(weights[i], capacity + 1):  # 正序！
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    return dp[capacity]
```

**0-1 背包 vs 完全背包的唯一区别**：内层循环方向不同！
- 0-1 背包：`for w in range(cap, weight-1, -1)`（倒序，每个物品只用一次）
- 完全背包：`for w in range(weight, cap+1)`（正序，每个物品可多次使用）

#### 最长子序列型

```python
# 最长递增子序列（LIS）
def length_of_LIS(nums):
    # dp[i] = 以 nums[i] 结尾的 LIS 长度
    dp = [1] * len(nums)
    for i in range(len(nums)):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)


# 最长公共子序列（LCS）
def longest_common_subsequence(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i - 1] == text2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    return dp[m][n]
```

#### 区间 DP

```python
# 最长回文子串
def longest_palindrome(s):
    n = len(s)
    dp = [[False] * n for _ in range(n)]
    start, max_len = 0, 1

    # 所有长度为 1 的都是回文
    for i in range(n):
        dp[i][i] = True

    # 长度为 2
    for i in range(n - 1):
        if s[i] == s[i + 1]:
            dp[i][i + 1] = True
            start, max_len = i, 2

    # 长度 ≥ 3
    for length in range(3, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            if s[i] == s[j] and dp[i + 1][j - 1]:
                dp[i][j] = True
                start, max_len = i, length

    return s[start:start + max_len]
```

### 6.3 DP 常见题型速查

| 类型 | 典型题 | 关键点 |
|------|--------|--------|
| 线性 DP | 爬楼梯、打家劫舍 | 一维数组递推 |
| 背包 DP | 0-1 背包、完全背包 | 一维优化、正/倒序 |
| 子序列 DP | LIS、LCS、编辑距离 | 二维 dp，注意初始化 |
| 区间 DP | 最长回文子串、石子合并 | 枚举长度+起点 |
| 状态机 DP | 股票买卖系列 | dp[i][状态] |
| 树形 DP | 打家劫舍 III | 后序遍历 + DP |
| 计数 DP | 不同路径 | dp 表含义是"方案数" |

## 专题 7：回溯算法

### 7.1 回溯三要素

```
1. 路径：已经做出的选择
2. 选择列表：当前可以做的选择
3. 结束条件：到达决策树底层，无法再做选择

模板：
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径.copy())
        return

    for 选择 in 选择列表:
        做选择
        backtrack(路径, 更新后的选择列表)
        撤销选择（回溯）
```

### 7.2 全排列

```python
def permute(nums):
    result = []
    used = [False] * len(nums)

    def backtrack(path):
        if len(path) == len(nums):
            result.append(path[:])  # 重要：复制 path！
            return

        for i in range(len(nums)):
            if used[i]:
                continue
            used[i] = True          # 做选择
            path.append(nums[i])
            backtrack(path)         # 递归
            path.pop()              # 撤销选择
            used[i] = False

    backtrack([])
    return result
```

### 7.3 子集

```python
def subsets(nums):
    result = []

    def backtrack(start, path):
        result.append(path[:])      # 每个节点都是一个子集

        for i in range(start, len(nums)):
            path.append(nums[i])
            backtrack(i + 1, path)  # 从 i+1 开始，不重复选
            path.pop()

    backtrack(0, [])
    return result
```

### 7.4 组合

```python
def combine(n, k):
    """从 1..n 中选 k 个数的所有组合"""
    result = []

    def backtrack(start, path):
        if len(path) == k:
            result.append(path[:])
            return

        # 剪枝：剩余的数不够凑满 k 个时提前结束
        for i in range(start, n - (k - len(path)) + 2):
            path.append(i)
            backtrack(i + 1, path)
            path.pop()

    backtrack(1, [])
    return result
```

## 专题 8：贪心算法

贪心没有固定模板，核心是**证明局部最优能推出全局最优**。

### 典型贪心题

**跳跃游戏**（能否跳到终点）：
```python
def can_jump(nums):
    max_reach = 0
    for i, jump in enumerate(nums):
        if i > max_reach:  # 当前位置不可达
            return False
        max_reach = max(max_reach, i + jump)
    return True
```

**区间调度**（最多不重叠区间数）：
```python
def max_intervals(intervals):
    # 按结束时间排序，每次选最早结束的
    intervals.sort(key=lambda x: x[1])
    count, last_end = 0, float('-inf')
    for start, end in intervals:
        if start >= last_end:
            count += 1
            last_end = end
    return count
```

## 专题 9：位运算技巧

### 核心技巧速查

```python
# 基础操作
x & 1                    # 判断奇偶（1=奇, 0=偶）
x >> 1                   # 除以 2
x << 1                   # 乘以 2
x & (x - 1)              # 消除最低位的 1
x & -x                   # 只保留最低位的 1
x ^ x                    # 0（自己异或自己得 0）
x ^ 0                    # x（任何数异或 0 不变）

# 常见应用
# 1. 找数组中唯一出现一次的数（其他都出现两次）
single = 0
for num in arr:
    single ^= num

# 2. 判断是否为 2 的幂
def is_power_of_two(n):
    return n > 0 and (n & (n - 1)) == 0

# 3. 计算二进制中 1 的个数
def count_ones(n):
    count = 0
    while n:
        n &= (n - 1)   # 每次消除最低位的 1
        count += 1
    return count

# 4. 枚举所有子集（位掩码法）
def all_subsets(nums):
    n = len(nums)
    result = []
    for mask in range(1 << n):      # 0 到 2^n - 1
        subset = [nums[i] for i in range(n) if mask & (1 << i)]
        result.append(subset)
    return result
```
